## Updating

Each project keeps a working copy of its legal docs in its own (private) repo. When a doc changes:

1. Edit the source-of-truth copy in the project repo
2. Copy the updated version into the matching subfolder here
3. Commit + push — GitHub Pages republishes within ~30 seconds

The two copies should match. If they drift, **this repo is the authoritative public version** (it's what users / app stores actually read).
